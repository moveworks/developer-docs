## Introduction

UKG Pro HCM (Human Capital Management) is UKG's comprehensive HR platform covering HR administration, benefits, talent acquisition, onboarding, compensation, and people analytics. It serves as the system of record for the full employee lifecycle. For full API documentation, refer to the [**UKG Pro HCM Developer Hub**](https://developer.ukg.com/hcm/reference/welcome-to-the-ukg-pro-api).
